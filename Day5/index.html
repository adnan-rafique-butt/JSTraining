~ cos ()

The Math.cos() method returns the cosine of a number. And Math.cos() returns a numeric value between -1 and 1, which represents the cosine of the angle.

~ exp ()

The Math.exp() method returns the value of Ex, where E is Euler's number (approximately 2.7183) and x is the number passed to it.

~ log ()

The Math.log() method returns the natural logarithm (base E) of a number.

~ max (x,y,z ..... n)

The Math.max() method returns the number with the highest value.

~ min (x,y,z ..... n)

The Math.min() method returns the number with the lowest value.

~ pow (x,y)

The Math.pow() method returns the value of x to the power of y.

~ ceil ()

The Math.ceil() method rounds a number UPWARDS to the nearest integer, and returns the result.

~ floor ()

The Math.floor() method rounds a number DOWNWARDS to the nearest integer, and returns the result.

~ round ()

The Math.round() method rounds a number to the nearest integer. Note: 2.49 will be rounded down (2), and 2.5 will be rounded up (3).

~ trunc ()

The Math.trunc() method Returns the integer part of x (new in ES6)

~ sin ()

The Math.sin() method returns the sine of a number. And Math.sin(x) returns a value between -1 and 1, which represents the sine of the parameter x.

~ tan ()

The Math.tan() method returns the tangent of a number.

~ sqrt ()

The Math.sqrt() method returns the square root of a number.
